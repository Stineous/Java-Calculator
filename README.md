Desktop Calculator App
A clean, responsive, and lightweight desktop calculator application built using Java and Java Swing. This application features an intuitive graphical user interface (GUI) with robust input validation and error-handling logic for everyday arithmetic operations.

🚀 Features
Core Arithmetic Operations: Supports addition, subtraction, multiplication, and division.

Smart Input Validation: Implements strict validation controls (e.g., preventing multiple consecutive decimal points within a single operand).

Robust Error Handling: Built-in safeguards against division by zero and malformed expressions.

Custom UI Theme: A polished, modern color palette using Swing layout managers for uniform button sizing and clear display visibility.

Event-Driven Architecture: Clean separation of UI components and mathematical logic using decoupled action listeners and conditional switch-case handling.
